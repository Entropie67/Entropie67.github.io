Exemple d’évolution d’un jeu vidéo : les SIMS

L’évolution des jeux vidéo évolue avec les progrès technologiques, c’est le cas pour le jeu vidéo les SIMS qui a évolué à travers le temps ayant quatre versions différentes. Le jeu vidéo les SIMS est un jeu de simulation de vie. Description du jeu les SIMS : Les règles du jeu sont extrêmement simples, il suffit de vous occuper des besoins de votre personnage prénommé « SIMS » en leur choisissant une identité, un lieu d’habitation… En outre vous devez lui créer une vie selon vos envies. Les quatre versions différentes : Les SIMS 1 sont sortis le 4 février en 2000. Les SIMS 2 sont sortis le 14 septembre en 2004. Les SIMS 3 sont sortis le 2 juin 2009. Les SIMS 4 sont sortis le 2 septembre 2014.
